# OtakuZoneFlix

Plataforma premium de streaming de animes com foco em venda de acessos por temporadas, inspirada na interface da Netflix.

## 🚀 Funcionalidades
- Interface moderna estilo Netflix.
- Venda de acesso por temporada.
- Sistema de pagamento manual com upload de comprovativo.
- Painel Admin para aprovação de pagamentos e gestão de catálogo.
- Player de vídeo com playlist de episódios.
- Integração com MongoDB Atlas e Cloudinary.

## 🛠️ Tecnologias
- **Frontend**: React (Vite)
- **Backend**: Node.js (Express)
- **Database**: MongoDB Atlas
- **Storage**: Cloudinary

## 📝 Como rodar
1. Clone o repositório.
2. Configure o `.env` no diretório `/server` (veja o walkthrough).
3. Instale dependências: `npm install` no client e server.
4. Rode as sementes: `node seed.js` no server.
5. Inicie: `npm run dev`.
